# Homeloan_Calculator

## Features

- Calculates home loan EMI based on loan amount, interest rate, and tenure.
- Supports yearly extra EMI and yearly hike percentage to simulate faster repayment.
- Displays a detailed amortization table with month-wise principal, interest, and remaining balance.
- Shows a visual principal vs. interest breakdown using an interactive pie chart.
- Provides a payment summary (total principal, total interest, total amount, EMI, total months, total years, and savings from extra EMI).
- Allows exporting the amortization schedule to PDF.
- Allows exporting the amortization schedule to Excel.

To use the calculator, simply open `Homeloan_Calculator.html` in a browser and adjust the inputs and sliders as needed.
